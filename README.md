# Task Manager

**Descripción:**
Task Manager es una aplicación sencilla y eficiente para gestionar tus tareas diarias. Con ella podrás crear, editar, eliminar tareas y marcar aquellas que hayas completado. Organiza tu trabajo de forma fácil y rápida.

## Características
- Crear, editar y eliminar tareas de manera intuitiva.
- Marcar tareas como completadas.
- Filtrar tareas por estado (pendientes, completadas).

## Instalación
Sigue estos pasos para clonar el repositorio y configurar el entorno de desarrollo:

1. Clona el repositorio del frontend:
   ```bash
   git clone https://github.com/camilo0999/task-manager.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd task-manager
   ```
3. Instala las dependencias necesarias:
   ```bash
   npm install
   ```

Para ejecutar la aplicación en un entorno local, usa el siguiente comando:
```bash
npm run dev
```

## Repositorio del Backend
Si deseas trabajar también en el backend, puedes encontrar el repositorio en el siguiente enlace:
[https://github.com/camilo0999/task-manager-backend](https://github.com/camilo0999/task-manager-backend)

## Uso
Una vez que la aplicación esté en funcionamiento, accede a ella a través de tu navegador en [http://localhost:5173](http://localhost:5173). En la interfaz podrás:

- Crear nuevas tareas.
- Editar tareas existentes.
- Eliminar tareas que ya no sean necesarias.
- Marcar tareas como completadas.
